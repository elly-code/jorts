# Backstory


## Early days

Initially, Jorts was called Notejot, and has been developed by Lainsce (https://github.com/lainsce/) in gtk3 for elementary OS, distributed as debian package in the old appcenter, before it was flatpak only.

Much of the design of the current application takes from that early era. The storage was done into a json file, the editablelabel was a custom widget, and there was a thumbtack button to keep notes always on top. The gear menu only allowed changing themes.

At some point, Lainsce decided she wanted Notejot to take another direction, and changed notejot to be a note-taking application instead of a sticky notes app


## ellie-commons

the ellie-commons community has been created in 2025 as a collective effort to update old unmaintained elementary OS applications

one of the projects people wanted to revive was Notejot as it once was. Thus a fork has been created by @UncleTallest (https://github.com/UncleTallest) of the latest commit of "Notejot as sticky-note app", going back to July 10, 2020

then Team Constellation (https://github.com/teamcons) came around and learned the ropes of Vala and GTK4 working on it.



## identity

The new name took a while. The goal was for something whimsy and not as generic as some variation of "notes". Charlie suggested Jorts, and explained: "It is neither jeans, nor shorts, just like jorts". It sounded fun and fresh

the icon and default color have been recolored blue to fit the name

At some point, wpkelso (https://github.com/wpkelso/) made a new icon for Jorts, fitting more modern standards

For Pride Month, wpkelso has also been commissionned to do a Pride variant of Jorts icon. The idea of a seasonal icon lingered around, and more were then done, to add in color, whimsy and surprise


## elly-code

at some point, some differences with the admins of ellie-commons caused a lot of applications to be moved to elly-code.
